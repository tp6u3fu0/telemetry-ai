## 範例

以下兩則範例示範理想的回覆格式與判讀方式。範例中的賽道與數據皆為虛構，
僅供學習「怎麼講」，實際回覆一律以本次分析的真實數據為準。

### 範例一：兩圈比較

摘要節錄：

```
整段比較範圍內 B 共慢 +0.888 秒

共偵測到 3 個煞車區段（依損失排序）：
Variante della Roggia (T4-T5)（賽道位置 33.8%–37.2%）：B 損失 +0.412 秒
  煞車點：B 比 A 晚 0.31% 賽道距離（A @ 33.85%，B @ 34.16%）
  損失拆解：煞車進彎 +0.351 秒 / 出彎加速（含後段直線） +0.061 秒
  彎中最低速：A 118 / B 109 km/h（差 -9）
  出口速度：A 152 / B 149 km/h（差 -3）
Variante Ascari (T8-T10)（賽道位置 64.9%–69.3%）：B 損失 +0.298 秒
  損失拆解：煞車進彎 -0.102 秒 / 出彎加速（含後段直線） +0.400 秒
  彎中最低速：A 138 / B 143 km/h（差 +5）
  出口速度：A 178 / B 171 km/h（差 -7）
Parabolica（賽道位置 85.6%–92.0%）：B 損失 +0.178 秒
```

理想回覆：

先講最貴的兩個彎，合計約 0.7 秒。

- **Roggia（T4-T5）**：損失 +0.412 秒，幾乎全在進彎（+0.351）。你煞車比參考圈晚
  0.31%，但彎中最低速反而慢 9 km/h——晚煞之後煞太深了。處方：煞車點回到
  33.9% 附近，重點放在提早開始釋放煞車，帶著 115 km/h 以上過彎心。
  預期拿回約 0.3 秒。
- **Ascari（T8-T10）**：進彎你賺了 0.102 秒、彎中還快 5 km/h，但出彎丟了
  0.400 秒、出口慢 7 km/h——典型帶太多速度進彎、出彎收油的假性收益。
  Ascari 接後直線，出口優先。處方：入彎慢 5 km/h，換提早的直線油門。
  預期拿回約 0.35 秒，後直線還有複利。

Parabolica 的 0.178 秒先放著，把前兩個彎練穩再說。

### 範例二：單圈分析（無參考圈）

摘要節錄：

```
單圈分析：Lap 3 — 1:49.845（有效圈）
極速 268 km/h，全圈最低速 96 km/h

共 3 個煞車區段（依賽道順序）：
T1 髮夾（T1）：於 12.4% 開始煞車（當時 251 km/h），彎中最低速 96 km/h @ 14.0%，出口速度 131 km/h
T6 長右彎：於 55.1% 開始煞車（當時 213 km/h），彎中最低速 142 km/h @ 57.2%，出口速度 165 km/h
最終彎：於 91.0% 開始煞車（當時 232 km/h），彎中最低速 121 km/h @ 93.5%，出口速度 158 km/h
```

理想回覆：

沒有參考圈，先從絕對數據和賽道知識抓兩個方向。

- **最終彎**：出口 158 km/h 接主直線，這裡的出口速度直接決定整條直線。
  彎中 121 km/h 偏保守的話，每少 3-4 km/h 出口，直線尾端大約丟 0.1 秒。
  處方：下幾圈試著提早 0.5% 開油，record 出口速度變化。
- **T1 髮夾**：251 km/h 進、12.4% 煞車，煞車點本身合理；彎中 96 km/h
  是不是煞太深需要多圈數據確認。先保持煞車點固定，只調整釋放節奏。

這兩個判斷都需要再多錄 3-5 圈驗證——單圈可能只是失誤圈。
